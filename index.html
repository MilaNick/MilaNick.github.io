Проект "Игры разума"

«Игры разума» — набор из пяти консольных игр. Каждая игра задает вопросы, на которые нужно дать правильные ответы.

Это первый проект, реализованный при изучении профессии Javascript-разработчика на hexlet.io.

Работа над проектом началась с настройки среды окружения: установлен node.js, прописаны зависимости в npm, для чтения пользовательского ввода подключена сторонняя библиотека readlineSync. Для удобства работы, был использован make. Для автоматического отслеживания качества кода - настроен линтер (Eslint), для непрерывной интеграции подключен сервис CodeClimate.

Для удобства был установлен linux, и в работе с терминалом закреплен теоретический материал по основам командной строки. Для записи игр в терминале был использован удобный инструмент asciinema. Проект также помог разобраться с системой контроля версий, одним из главных инструментов в арсенале любого разработчика.
